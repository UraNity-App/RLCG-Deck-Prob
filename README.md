# その他設定
・敵の入れ替えはしないものとする<br>
・交換所の出現確率は**一律27.1％(訂正。敵１枠あたり約10%)**、ただし使用したら次の３階層は抽選から外れる<br>
・交換所の枠数は21枠、交換所ガチャはコンプリート済みとする<br>
・ナイトメア強化を受ける確率は、15階層から22階層は50％、23階層以降は100％とする<br>
・ナイトメア強化の判定は簡単のためカード毎に行う<br>
・同じ敵は２階層連続で出ないようになっているが、これを考慮してもよほど特殊な場合でなければ確率に大きく影響しないのでこの仕様は無視する<br>
・骸の異形から通常敵のカードを取得することは考慮していない<br>
